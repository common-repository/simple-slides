== Description ==

**Simple Slides in undergoing big updates, version 2.0 will be out soon!**

Create slideshows that uses the world's most awesome slider, the [Nivo Slider](http://nivo.dev7studios.com/ "The world's most awesome jQuery & WordPress Image Slider"). Uses shortcodes with a number of attributes to allow customized embedding in widgets, posts, pages, or theme files. Supports the following browsers: Internet Explorer v7+, Firefox v3+, Google Chrome v4+, Safari v4+, and Opera v10+. **Please don't hesitate to contact us for questions, suggestions, etc.** For more information check the [plugin documentation](http://apocalypseboy.com/simple-slides/ "Simple Slides for Wordpress").

== Installation ==

1. Upload `simple-slides.zip` to the `/wp-content/plugins/` directory and unarchive.
1. Activate the plugin through the 'Plugins' menu in WordPress.

Or

1. Download it through the WordPress admin and activate.
